One-Time Setup Guide

Connect your product catalogue to Google Sheets in under 5 minutes — free, forever.

1

Create the Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it AI Products Leads.

In Row 1, add these column headers exactly:

Timestamp  |  Product  |  Name  |  Phone  |  Email  |  Message

Each new lead will appear as a new row below this header automatically.

2

Open Apps Script

Inside your Google Sheet, click Extensions → Apps Script.

Delete any code in the editor and paste the following script in full:

// Google Apps Script — paste this in full
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data  = JSON.parse(e.postData.contents);

  sheet.appendRow([
    data.timestamp || new Date().toLocaleString('en-IN'),
    data.product  || '',
    data.name     || '',
    data.phone    || '',
    data.email    || '',
    data.message  || '',
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({ status: 'ok' }))
    .setMimeType(ContentService.MimeType.JSON);
}
3

Deploy as a Web App

In Apps Script, click Deploy → New deployment.

  • Type: Web app
  • Description: Lead capture for AI Products
  • Execute as: Me
  • Who has access: Anyone

Click Deploy. Authorise when prompted (click "Advanced → Go to…" if Google shows a warning — this is your own script).

Copy the Web App URL — it looks like:
https://script.google.com/macros/s/AKfycb.../exec
You'll need this in the next step.
4

Paste the URL into app.js

Open app.js and find line 6 at the very top:

const SHEET_ENDPOINT = 'YOUR_GOOGLE_APPS_SCRIPT_URL';

Replace YOUR_GOOGLE_APPS_SCRIPT_URL with the URL you just copied:

const SHEET_ENDPOINT = 'https://script.google.com/macros/s/AKfycb.../exec';

Also replace 91XXXXXXXXXX on line 7 with your actual WhatsApp number (no spaces, include country code).

5

Commit & push to GitHub

Save app.js, then run:

git add app.js
git commit -m "Connect lead form to Google Sheets"
git push origin main
✓ You're done. Every time someone fills the form, a new row appears in your Google Sheet with their name, phone, email, product interest, and message — timestamped and sorted automatically.
